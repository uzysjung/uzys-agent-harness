**Always (자동 실행)**:
- 산출물 완성 후 reviewer 검증 실행
- 수치/날짜/법규 인용 시 1차 출처 확인
- 근거 불충분 시 명시 ("근거 불충분" 라벨링)
- 변경 이력 git으로 관리

**Ask First (확인 후 실행)**:
- 최종 문서 포맷 선택 (pptx vs docx vs pdf)
- 외부 데이터 소스 참조 (API 호출, 웹 검색)
- 민감한 재무/전략 정보 포함 여부
- main 브랜치 머지

**Never (금지)**:
- 검증되지 않은 수치를 사실처럼 제시
- 출처 없는 통계/시장 데이터 사용
- 민감 정보(NDA 대상 등) 커밋
